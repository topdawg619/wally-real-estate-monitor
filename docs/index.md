# 🏠 Wally Real Estate Monitor

**Open-source real estate monitoring system for tracking properties across multiple listing sites**

## Quick Demo

Try the [Live Demo](https://topdawg619.github.io/wally-real-estate-monitor) (GitHub Pages)

## Features

- 🕷️ **Multi-site scraping** of Redfin, Zillow, and Realtor.com
- 📱 **Mobile-responsive dashboard** for daily property checking  
- ⚡ **Automated monitoring** with 3x daily scans
- 📊 **Price change tracking** with historical data
- ❤️ **Favorites system** with notes and bookmarks
- 📈 **Market statistics** and trend analysis
- 🎯 **Smart filtering** by ZIP code, price, source, and more

## Quick Start

```bash
git clone https://github.com/topdawg619/wally-real-estate-monitor.git
cd wally-real-estate-monitor
python setup.py setup
python setup.py dashboard
```

Opens at http://localhost:8080

## Target Market

Initially built for **Chula Vista, CA** (ZIP code 91913) and surrounding areas:
- 91913 - Chula Vista East ($857K median)
- 91910 - Chula Vista Central ($790K median)  
- 91911 - Chula Vista West ($725K median)
- 91915 - Bonita ($833K median)
- 91917 - National City

**Easily configurable for any market** - just update the ZIP codes!
